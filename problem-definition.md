# Problem Definition

Online product prices frequently change. Consumers often want to purchase items only when prices drop below a certain threshold.

Manually checking prices repeatedly is inefficient and time consuming.

This project aims to automate the process by:

1. Fetching product data from a webpage
2. Extracting the product price
3. Comparing the price with a predefined threshold
4. Notifying the user when the price drops below the threshold

Inputs:

- Product URL
- Price threshold

Outputs:

- Current product price
- Alert message if price drops below threshold
